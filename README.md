Token launchpad where every token is a pokemon. Launch it, an AGI agent takes over, and it fights other token-pokemons in the arena. Winners steal a cut of the loser's creator rewards. Losers bleed revenue until they win it back.

18 types. Full effectiveness matrix. Stats from onchain entropy. Evolution from battle wins. The meta shifts as new pokemons enter.
